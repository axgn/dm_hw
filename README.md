# Data Mining Homework

数据集参考下载地址：
<https://grouplens.org/datasets/movielens/>

- machinelearning 文件夹里存放有关机器学习相关的电影推荐实现
- deeplearning 文件夹里存放有关深度学习相关的电影推荐实现
- datasets目录用来存放相关数据集
- reference目录用来存放相关参考

使用`uv sync`可以直接一键创建虚拟环境，并安装相关依赖，然后把数据集下载下来放入dataset目录下，然后使用uv创建的虚拟环境来运行jupyter notebook文件就可以了。
uv的安装参考<https://uv.doczh.com/getting-started/installation/>
