# 电影推荐系统的相关机器学习实现

testdata.ipynb中记录了对于数据集的基本查看。

train.ipynb 为第一次训练的过程，其中没有测试集和训练集的划分，只是对除了自身的所有数据进行相似度查询，然后得到结果

train2.ipynb 使用留出法得到训练集和测试集并进行了相关的计算。

train3.ipynb deepseek写的一整个训练的过程，但是和前面使用的数据集量级是不同的，使用的是1m的数据集，代码没有细看，仅作为测试（以及无法完成任务时的后备方案）

train4.ipynb 和train2相比使用的划分数据集的方法是不一致的，在train4中使用用户进行划分，而在train2中是整体上进行划分。
